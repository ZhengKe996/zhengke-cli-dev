# Zhengke-Cli-Dev

一个基于 Node.js 的前端 cli 脚手架，偷懒小工具（功能不断完善 ing...）

### 安装

```bash
npm i -g @zhengke-cli-dev/core
```

### 使用

1. 使用命令行窗口 进入指定文件夹
2. 执行命令 `zhengke-cli-dev init`
3. 选择 初始化类型 项目(Project) or 组件(Component)
4. 请输入 项目 or 组件 名称
5. 请输入 项目 or 组件 版本号
6. 请选择 项目 or 组件 模板

一切将会自动进行...
